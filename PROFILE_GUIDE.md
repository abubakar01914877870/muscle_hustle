# User Profile Guide - Fitness Tracking Application

## Profile Features

The user profile system collects comprehensive fitness-related information to provide personalized tracking and recommendations.

## Profile Sections

### 1. Personal Information
- **Full Name**: User's complete name
- **Email**: Account email (username)
- **Date of Birth**: Used to calculate age
- **Age**: Automatically calculated from date of birth
- **Gender**: Male, Female, or Other
- **Phone**: Contact number

### 2. Physical Measurements
- **Height**: In centimeters (cm)
- **Current Weight**: In kilograms (kg)
- **Target Weight**: Goal weight in kilograms (kg)
- **BMI**: Body Mass Index - automatically calculated from height and weight

### 3. Fitness Information
- **Fitness Level**: 
  - Beginner
  - Intermediate
  - Advanced
  
- **Fitness Goal**:
  - Weight Loss
  - Muscle Gain
  - Maintenance
  - Endurance
  
- **Activity Level**:
  - Sedentary (little or no exercise)
  - Lightly Active (exercise 1-3 days/week)
  - Moderately Active (exercise 3-5 days/week)
  - Very Active (exercise 6-7 days/week)
  - Extremely Active (physical job or training twice per day)
  
- **Workout Frequency**: Number of workout days per week (0-7)
- **Preferred Workout Time**: Morning, Afternoon, or Evening

### 4. Health Information
- **Medical Conditions**: Any health conditions that may affect fitness activities
- **Dietary Restrictions**: Allergies, preferences, or dietary requirements

## Accessing Your Profile

1. **View Profile**: Click "Profile" in the navigation menu
2. **Edit Profile**: Click "Edit Profile" button on your profile page
3. **Change Password**: Click "Change Password" button on your profile page

## Profile Features

### View Profile
- See all your information in an organized layout
- View calculated metrics (Age, BMI)
- Quick access to edit and password change

### Edit Profile
- Update any profile information
- All fields are optional
- Changes are saved immediately
- Organized into sections for easy navigation

### Change Password
- Requires current password for security
- New password must be at least 6 characters
- Confirmation required to prevent typos

## Tips

- Complete your profile for better personalized recommendations
- Update your weight regularly to track progress
- Set realistic target weights based on your fitness goals
- Be honest about activity level for accurate calculations
- Keep medical conditions and dietary restrictions up to date

## Privacy

- Your profile information is private and secure
- Only you can view and edit your profile
- Admin users can view user lists but not detailed profile information
