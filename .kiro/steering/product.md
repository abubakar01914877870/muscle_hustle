# Product Overview

Muscle Hustle is a Flask-based fitness tracking web application that helps users manage their fitness journey through comprehensive profile management, exercise tracking, and progress monitoring.

## Core Features

- User authentication with role-based access (Admin/User)
- Comprehensive fitness profiles with BMI calculation, goals, and health information
- Exercise library with detailed instructions, media support (images/videos), and filtering
- Progress tracking with photos and measurements
- Admin dashboard for user management
- Image storage using base64 encoding in MongoDB

## Target Users

- Fitness enthusiasts tracking their workout progress
- Personal trainers managing client profiles
- Gym administrators overseeing user accounts
